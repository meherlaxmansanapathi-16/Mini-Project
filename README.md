# Online Exam Evaluation System

An Online Exam Evaluation System developed as a mini project to conduct quizzes, evaluate answers automatically, and display results efficiently.

> **Note:** The project was initially developed using **C++** for the backend. It is currently being migrated to **Python (Flask)** to build a more scalable and user-friendly web application.

---

## Features

- User Login System
- Student Dashboard
- Faculty/Admin Dashboard
- Online MCQ Examination
- Automatic Score Calculation
- Result Display
- Simple and Interactive User Interface
- Secure Backend Processing

## Installation

Clone the repository:

```bash
git clone https://github.com/meherlaxmansanapathi-16/Mini-Project.git
```

Install the required packages:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python app.py
```

Open your browser and visit:

```
http://127.0.0.1:5000
```
